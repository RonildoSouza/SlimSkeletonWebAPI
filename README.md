# Slim Skeleton Web API

This project is a skeleton for build your web API using:

* [Slim](https://www.slimframework.com/)
* [Doctrine/ORM](https://www.doctrine-project.org/)
* [Monolog](https://github.com/Seldaek/monolog/blob/master/README.md)
* [AutoMapper+](https://github.com/mark-gerarts/automapper-plus/blob/master/README.md)
* [Swagger-PHP](http://zircote.com/swagger-php/)
